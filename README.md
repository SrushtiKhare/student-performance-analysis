# Student Performance Analysis

## 📊 Overview
This project analyzes student exam performance using Python.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📈 Insights
- Students with standard lunch perform better
- Higher parental education leads to better scores
- Female students slightly score higher

## 📁 Dataset
StudentsPerformance dataset
