## Project Overview

This project analyzes student academic performance using Python and Exploratory Data Analysis (EDA). The objective is to understand how different factors such as gender, parental education, lunch type, and test preparation influence students' academic scores.

The analysis was performed using **Pandas** for data manipulation and **Matplotlib** for data visualization.

---

## Objectives

- Analyze student performance data.
- Compare average scores across different student groups.
- Identify factors affecting academic performance.
- Visualize trends using charts.
- Practice Exploratory Data Analysis (EDA) using Python.

## Key Insights

- Students with **standard lunch** scored higher on average.
- Students whose parents had **higher education levels** generally performed better.
- Female students achieved slightly higher average scores than male students.
